## 🚀 Quick Start
```bash
# Backend (Terminal 1)
cd backend
.\run-backend.bat

# Frontend (Terminal 2)
cd frontend
npm install  # first time only
npm run dev

# Open: http://localhost:5173
```

## 📂 Key Files to Customize
1. `frontend/src/utils/constants.js` - Update contact info, phone, email
2. `frontend/src/pages/Home.jsx` - Update carousel images (lines 9-30)
3. `frontend/src/pages/Contact.jsx` - Update Google Maps (line 150)
